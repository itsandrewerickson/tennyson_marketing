# Context

The Offer Memorandum and landing page need to look professional and match institutional quality (Colliers, CBRE, Marcus & Millichap) while incorporating Next Wave Group branding.

# Decision

## Visual Style
Match the Colliers OM example with these adaptations:

### Color Palette
| Use | Color | Hex |
|-----|-------|-----|
| Primary Dark | Charcoal | #2D3436 |
| Accent | Gold | #C9A227 |
| Background | Cream | #F5F5F0 |
| Background Alt | White | #FFFFFF |
| Text Primary | Dark Gray | #1A1A1A |
| Text Secondary | Medium Gray | #666666 |

### Typography
- **Headings**: Inter Bold or Playfair Display (serif option)
- **Body**: Inter Regular
- **Data/Tables**: Inter Medium
- **Fallback**: System fonts

### Layout Principles
- **PDF**: Landscape orientation, spread format (2-page view)
- **Landing Page**: Single column, mobile-first
- **Whitespace**: Generous margins, avoid cramped layouts
- **Photos**: Full-bleed where possible, high contrast

### Component Patterns
- **Headers**: Dark bar with property name, gold accent line
- **Section Titles**: Serif font, gold underline
- **Data Tables**: Alternating row colors, clean borders
- **Call-to-Action**: Gold button on dark background
- **Photo Captions**: Small caps, light gray

## Branding Integration
- Next Wave Group logo in footer (both deliverables)
- Logo in PDF cover and contact page
- Consistent with any existing Next Wave brand guidelines

# Consequences

**Positive:**
- Professional appearance matching institutional quality
- Consistent look across PDF and landing page
- Flexible enough for Next Wave branding overlay

**Negative:**
- Requires high-quality photography to look good
- Dark headers may not print well (consider print version)
- Custom fonts may slow page load slightly
